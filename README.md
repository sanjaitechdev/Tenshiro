# Tenshiro AI

A production-ready, accessible, and AI-powered Student Career Navigation Platform.

## Mission
Help students become job-ready before graduation through structured Domain -> Role -> Roadmap -> AI Mentorship -> Placement Execution -> Dashboard Tracking system.

## Tech Stack
- **Frontend**: React (Vite), Pure CSS
- **Backend**: Node.js, Express.js
- **Database**: MySQL
- **AI**: OpenAI API

## Set Up
1.  Clone the repository.
2.  Install dependencies:
    -   Frontend: `cd frontend && npm install`
    -   Backend: `cd backend && npm install`
3.  Configure `.env` file (see `.env.example`).
4.  Run the application:
    -   Frontend: `npm run dev`
    -   Backend: `npm start`
